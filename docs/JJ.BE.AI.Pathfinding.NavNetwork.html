Class: NavNetwork

JJ.BE.AI.Pathfinding.NavNetwork

The NavNetwork is a network of interconnected nodes representing world locations accessible by navigation. Each accessible locations contains a list of links to other nodes in the network, which can be used to navigate through the entirety of the network.
